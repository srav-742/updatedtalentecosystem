# TalentEcosystem

**TalentEcosystem** is an advanced, AI-powered recruitment platform designed to modernize the hiring process. It connects Job Seekers and Recruiters through a transparent, gamified, and intelligent interface.

### For Job Seekers
*   **AI Resume Analysis**: Get instant feedback on your resume's match percentage against specific jobs.
*   **Skill Assessments**: Take dynamic MCQ and Coding challenges generated by AI (Gemini/DeepSeek) to prove your skills.
*   **Mock Interviews**: Practice with AI-generated interview questions and get real-time scoring.
*   **Gamified Economy**: Earn coins for high scores and profile completion to unlock premium features.

### For Recruiters
*   **Smart Job Posting**: Define exact skill requirements.
*   **Automated Vetting**: Receive applications that are already screened, scored, and ranked.
*   **Dashboard**: Manage jobs and applicants in one place.

## ðŸ“¦ Setup & Run

1.  **Install Dependencies**
    ```bash
    npm install
    cd backend && npm install
    ```

2.  **Environment Variables**
    *   Create `.env` in `backend/` with:
        *   `MONGODB_URI`
        *   `GEMINI_API_KEY`
        *   `DEEPSEEK_API_KEY`
        *   `PORT=5000`

3.  **Run Application**
    *   **Backend**:
        ```bash
        cd backend
        node server.js
        ```
    *   **Frontend**:
        ```bash
        npm run dev
        ```
